\begin{center}
	\section{\problembzh}
\end{center}

\subsection{【问题描述】}
输入正整数$n$，求$\sum\limits_{1 \leq i < j \leq n} \gcd(i, j)$。

\subsection{【输入格式】}
从文件~\problemb.in~中读入数据。

输入包含多组数据，每组数据占一行，包含一个正整数$n$。

输入结束的标志为$n = 0$。

\subsection{【输出格式】}
输出到文件~\problemb.out~中。

对于每组数据，输出一行，即所求和。

答案对$10 ^ 9 + 7$取模。

\subsection{【样例输入】}
{\ttfamily
10

100

200000

0
}
\subsection{【样例输出】}
{\ttfamily
67

13015

143295493160
}

\subsection{【数据规模与约定】}
对于$100\%$的数据，输入包含不超过100组数据，$2 \leq n \leq 4 * 10 ^ 6$。
